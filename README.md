# diyticketer
Files, general information, and updates surrounding my DIY Ticketer Project

# About this project
As some of you may know, I'm a bus spotter from Greenhithe that is surrounded by outer-London buses back home. All of them use either the Ticketer TK100 or TK200. They include a 8-inch touchscreen display (resistive and capacitive for the respecitve models), an NFC reader, a reciept printer, and a camera for scanning tickets via QR code (although some operators don't spec this out). Getting a hold of one is very complicated, and even when you do get your hands on an ex-service one, they are extremely hard to do anything with since they are all locked to their respective operators, and since they have not been properly reverse engineered (except for maybe the TK200 since it's based on a Samsung Galaxy Tab Active), there is no custom firmware (CFW) that can be flashed onto these machines - from an operators perspective, and as a crude example, Arriva just can't go ahead and buy a Rotala Diamond-registered machine and use it on their buses since nothing would properly line up.

That was when I thought - what if I built my own one that runs on Linux or Windows 11, that can be used as both a Ticketer and an x64 desktop machine too? Similar ticketing machines already exist for stuff liike events or places, but not for buses. What if I could fill that gap and, hopefully, be the first one to do it? Well, considering my extensive knowlegde of computers and hardware, I've decided to take this project seriously!

# System Information
* Base system: Lenovo ThinkCentre M73 Tiny
* Storage: 1TB Seagate Mobile 5200RPM Mechanical HDD
* RAM: 8GB of DDR3 2400
* GPU: Intel HD Graphics 650
* CPU: Some 3rd gen Intel Core i3 running at 3ghz flat
* OS: Microsoft Windows 11 Enterprise Internet of Things LTSC 2024 (will probably change to Linux later down the line)
